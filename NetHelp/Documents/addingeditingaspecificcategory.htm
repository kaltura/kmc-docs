Adding/Editing a Specific Category

To add a category

1.  Select the Content tab and then select the Categories tab.

2.  Select Add Category.

The New Category window is displayed.

 

3.  Select No Parent if you want the category to be at the top level or select the parent category under which the new category should be created and select Next. We recommend that you minimize categories at the top level. The New Category window is displayed.

4.  Add metadata and click Save & Close.
