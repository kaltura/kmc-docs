Editing a Player

The changes you make to an existing player will propagate to all sites where the player has been embedded, including syndicated players on other sites.

To edit a player

1.  Select the Studio tab and slect Flash Studio.

2.  Click on the relevant player in the Player List.

Players created manually through Kaltura's professional services cannot be edited in this area.

3.  Select Edit from the Select Action drop down.


The player configuration Features tab is displayed. See Player Control Options.

4.  Check the control options for the player and then Save Changes.
