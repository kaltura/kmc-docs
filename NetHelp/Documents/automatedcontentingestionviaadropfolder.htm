Using a Drop Folder

The Kaltura Drop Folders are used to automate the ingestion of content into the Kaltura platform. The Drop Folder service constantly watches each drop folder for new content, and activates automatic ingestion of new content to the specific account.

Kaltura offers multiple configuration options for setting each drop folder to a specific workflow. To learn more read about Drop Folders see Kaltura Drop Folder Service for Content Ingestion.
