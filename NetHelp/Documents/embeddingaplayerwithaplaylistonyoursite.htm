Embedding a Player with a Playlist on your Site

After you've created a playlist, and have assigned the playlist to a player, see Designing and Configuring A Player you can publish the player to your site.

To embed a player with a playlist on your site

1.  Go to the Content tab and select the Playlists tab.

2.  Locate the playlist you want to publish on your website and click Preview & Embed.

The Embedding window is displayed.

3.  Select your preferred player from the drop-down menu. See Configuring the Player Features.

4.  (Optional) – Click Show Advanced Options.

a.  Select the Dehref="exportinganalyticstoacsvfile.htm">Exporting y Type. See Exporting ysettings.htm">Video Dehref="exportinganalyticstoacsvfile.htm">Exporting y Settings.

b.  Select the Embed Type. See Embed Code Types.

5.  (Optional) Check Include Search Engine Optimization data. See Search Engine Optimization.

6.  (Optional) Check Support for HTTPS embed code. See Support for HTTPS Embed Code.

7.  (Optional) Scan the QR code to preview the content in your mobile device.

8.  (Optional) Select to view a standalone page with this player by clicking on the provided link.

9.  Click Copy Embed Code and Close.

Copy Embed and Close is identical to the Copy embed code, except that Copy Embed and close closes the Preview and Embed window.
