Viewing an Entry with Captions in a Player

  To view the captions in a player

1.   Select the entry that you added the captions file to.

2.   Click Preview and Embed.

3.   In the Player window select the player to use to display this entry.

4.   Use the URL indicated in the View as a standalone page with this player.
