Kaltura Custom Metadata Functionality

You can create a unique metadata schema. The entire schema management is done via the KMC UI.  You can add fields of the following types:

•      Text field – values are free text

•      Text select list – this is similar to the text field, also known as List of Values, This option allows the publisher to define a predefined list of fields to be used. After you enter the values, you are presented with a select box element to select a value from the existing list (for example countries, months)

•      Date - a date field

•      Entry ID list - a list to a different entry (asset) that can be used to create compound structure. Examples of this would be, Related Videos, linking trailer to full-length film etc.
