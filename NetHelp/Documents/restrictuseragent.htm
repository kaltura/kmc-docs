Restrict User Agent

Use to block the player to specific user agents.

NOTE: Use these settings for the player display only. For general purpose access controls see entry level access controls.

 

To restrict the User Agent

1.  Select the Universal Studio tab and then select or create a player.

2.  Select the Plugins icon.

3.  Check the box next to the option.

4.  Enter the Restricted user agents. Enter a comma-separated list of browsers to search for.

5.  Enter the Restricted user agent title (error title).

6.  Enter the Restricted user agent message (error message).
