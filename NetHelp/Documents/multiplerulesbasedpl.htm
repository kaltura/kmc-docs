Multiple Rules Based Playlist - Advanced Mode

Use the Advanced Mode to create playlists based on multiple rules.

  To create/add a multiple rules based playlist

1.   Select the Content tab and then select the Playlists tab.

2.   Select Add Rules Based Playlist.

3.   Click Switch to Advanced Mode.

4.   Enter a Name for your playlist (required).

5.   Click Add Rule to create a new playlist rule. (See Creating a Rules Based Playlist  to learn about creating rules.)

6.   Add additional rules.

7.   Use the Remove Rule and Edit Rule buttons to make changes to your list of rules.

Rules are applied to the playlist in the order they appear. Use the arrows to change the order of the rules.
