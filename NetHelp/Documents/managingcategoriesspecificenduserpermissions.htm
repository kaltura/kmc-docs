Managing Categories’ Specific End-User Permissions

To manage categories specific end-user permissions

1.  Set the Entitlement Settings.

The Specific End-User Permissions window is displayed.

2.  Click Add Users.

3.  Select the Permission Level. See Specific End-Users Permissions.

4.  Select the Update Method. See End User Permission Attributes.

5.  Select End Users. Enter user names.

6.  Click Save & Close.
