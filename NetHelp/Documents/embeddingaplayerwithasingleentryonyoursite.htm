Embedding a Player with a Single Entry on your Site

To embed a player on your site

1.  Create a Player. See Configuring the Player Basics Settings.

2.  Select content (an entry) for your player. See Adding Content to a Player.
The Entries table from the Content tab is displayed.

3.  Click Preview and Embed.
The Embedding window is displayed.

4.  Select your preferred player from the drop-down menu. See Configuring the Player Features.

5.  (Optional) – Click Show Advanced Options.

a.  Select the Dehref="exportinganalyticstoacsvfile.htm">Exporting y Type. See Exporting ysettings.htm">Video Dehref="exportinganalyticstoacsvfile.htm">Exporting y Settings.

b.  Select the Embed Type. See Embed Code Types.

6.  (Optional) Check Include Search Engine Optimization data. See Search Engine Optimization.

7.  (Optional) Check Support for HTTPS embed code. See Support for HTTPS Embed Code.

8.  Preview: (Optional) Scan the QR code to preview the content in your mobile device.

9.  Preview: (Optional) Select to view a standalone page with this player by clicking on the provided link.

NOTE: The (Preview) QR code and standalone links are for preview use only. Do not use these links for production purposes.

 

10.          Click Copy Embed Code and Close.

Copy Embed and Close is identical to the Copy embed code, except that Copy Embed and close closes the Preview and Embed window.
