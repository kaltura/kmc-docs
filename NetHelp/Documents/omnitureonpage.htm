Omniture on Page

The Omniture s_code config version of the plugin allows you to connect the Omniture plugin to your existing s_code.js configuration for easy integration of video analytics into an Omniture site.

 

Field

Attribute

Value

Description

Code URL

s_codeUrl

resources/adobe_sample

The URL to the Ominture generated sCode file. If null, a local copy of s_code.js is used. Must be set in uiConf not via flashvar

Entry code name

s_codeVarName

. “s’ by default.

The name of the s-code entry point in the global window scope

Monitor event tracking interval

monitorEventInterval

10

Set to interval in seconds for tracking the Omniture m\:monitor” events.

Omniture events function name

trackEventMonitor

omnitureTrackingLog

A global callback function for logging Omniture events.

Media name concatenation rules

concatMediaName

null

A per partner key for special media concatenation rules. Default should be left null.

Kaltura player events

customEvents

openFullScreen,closeFull

A comma separated list of Kaltura events you want to track.

Omniture variables and properties

additionalEvarsAndProps

eVar51,prop44

A comma separated list of evars and props you want to pass along with every media event.

Kaltura version

additionalEvarsAndPropsValues

{mediaProxy.entry.creatorId},{mediaProxy.entry.created

A comma separated list of Kaltura values to pass along with every media event.
