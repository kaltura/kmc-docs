Editing a Player

All changes you make to an existing player will propagate to all sites where the player has been embedded, including syndicated players on other sites that have been embedded with an iframe or JavaScirpt embed. Direct references to the player SWFvia Flash object tags will not be updated to the HTML5 v2 players. Object tag embeds will need to be replaced with iframe embeds to support the Kaltura v2 players.

To edit a player

1.  Select the Universal Studio tab.

2.  Click on the relevant player in the Player List.

3.  Select an icon to modify the current player configuration.

4.  Click Save Player Settings.
